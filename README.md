## 1. Designing Machine Learning Systems

**Author:** Chip Huyen &nbsp;|&nbsp; **Year:** 2022 &nbsp;|&nbsp; **Publisher:** O'Reilly Media

### 📖 About
ML systems are complex because they involve many components and stakeholders. 
This book teaches you a practical approach to building ML systems that are 
reliable, scalable, and easy to maintain in real-world environments.

### 👥 Who Should Read This
- ML Engineers building production systems
- Data Scientists moving beyond notebooks
- Engineering Managers overseeing ML teams

### ✅ What You Will Learn
- How to design end-to-end ML pipelines
- Feature engineering and data management
- Model deployment, monitoring and debugging
- MLOps best practices for teams

### ⭐ What People Say
> *"The very best book you can read about how to build, deploy, and scale 
ML models at a company."* — Josh Wills, Former Director of Data Engineering, Slack

> *"If you are serious about ML in production, this book is essential."* 
— Laurence Moroney, AI & ML Lead, Google

## 2. AI Engineering

**Author:** Chip Huyen &nbsp;|&nbsp; **Publisher:** O'Reilly Media

### 📖 About
A practical guide to building applications powered by foundation models and 
large language models. Covers the full lifecycle from prototyping to production.

### 👥 Who Should Read This
- Developers building AI-powered applications
- ML Engineers working with LLMs and foundation models
- Anyone transitioning into AI engineering

### ✅ What You Will Learn
- Working with foundation models and APIs
- Prompt engineering and fine-tuning
- Evaluation and deployment of AI systems
- Cost, latency, and quality trade-offs

## 3. Deep Learning from Scratch

**Author:** Seth Weidman &nbsp;|&nbsp; **Publisher:** O'Reilly Media

### 📖 About
Builds deep learning concepts from the ground up using only Python and NumPy. 
No heavy frameworks — just pure first principles thinking.

### 👥 Who Should Read This
- Beginners who want to truly understand deep learning
- Students learning neural networks from scratch
- Anyone who wants to go beyond just using libraries

### ✅ What You Will Learn
- How neural networks actually work inside
- Backpropagation step by step
- Building layers, losses and optimizers from scratch
- Moving from basics to CNNs and RNNs

## 4. Hands-On Large Language Models

**Author:** Jay Alammar & Maarten Grootendorst &nbsp;|&nbsp; **Publisher:** O'Reilly Media

### 📖 About
A hands-on guide to understanding and using large language models. 
Covers both the theory and practical usage of modern LLMs with clear 
visuals and code examples.

### 👥 Who Should Read This
- Developers working with LLMs and NLP
- AI Engineers building text-based applications
- Anyone curious about how ChatGPT-like models work

### ✅ What You Will Learn
- How LLMs understand and generate language
- Text classification, embeddings and semantic search
- Fine-tuning and prompt engineering
- Building real-world NLP applications

## 5. Hands-On ML with Scikit-Learn Keras & TensorFlow

**Author:** Aurélien Géron &nbsp;|&nbsp; **Publisher:** O'Reilly Media

### 📖 About
One of the most popular ML books ever written. Covers everything from 
basic ML algorithms to deep neural networks with practical code 
examples throughout.

### 👥 Who Should Read This
- Complete beginners to ML and Deep Learning
- Python developers entering the AI field
- Students preparing for ML interviews

### ✅ What You Will Learn
- Core ML algorithms — regression, classification, clustering
- Neural networks with Keras and TensorFlow
- Computer Vision and NLP basics
- Training and deploying models
